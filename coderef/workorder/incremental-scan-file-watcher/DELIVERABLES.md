## Usage

```bash
# First scan - creates cache baseline
npx coderef-scan . --incremental

# Subsequent scans - only changed files
npx coderef-scan . --incremental

# Clear cache and rescan
rm .coderef/incremental-cache.json
npx coderef-scan . --incremental
```

---

## Implementation Notes

- Cache stored at `.coderef/incremental-cache.json`
- Tracks SHA-256 file hashes, timestamps, and sizes
- Recursive scans exclude cache to prevent double-updates
- Deleted files automatically removed from cache
- Verbose mode shows hit ratio and cache statistics
